# Подготовка Яндекс Директа для metallomsk.ru

## 1. Счетчик Яндекс Метрики

1. Создать счетчик для `metallomsk.ru`.
2. Включить Вебвизор, карту скроллинга, аналитику форм и автоматические цели.
3. Включить прием данных только с указанных адресов сайта.
4. Передать разработчику числовой ID счетчика для установки в `assets/site.js`.
5. Привязать этот счетчик к каждой кампании Директа.

## 2. Цели

Главные цели для оптимизации кампаний:

- `cart_send` — успешно отправлена заявка из корзины;
- `form_submit` — успешно отправлена форма расчета.

Дополнительные цели для анализа:

- `click_phone` — клик по телефону;
- `click_whatsapp` — переход в WhatsApp;
- `click_telegram` — переход в Telegram;
- `click_max` — переход в MAX;
- `click_email` — клик по электронной почте;
- `cart_add` — товар добавлен в заявку;
- `cart_open` — открыта корзина;
- `click_calc` — переход в калькулятор;
- `calculator_use` — пользователь изменил параметры калькулятора;
- `download_price` — скачивание файла с ценами.

Для стратегии сначала использовать только `cart_send` и `form_submit` как реальные обращения. Остальные цели нужны для диагностики воронки и не должны считаться равноценными заявке.

## 4. Параметры URL

Добавить в кампании общие параметры URL:

```text
utm_source=yandex&utm_medium=cpc&utm_campaign={campaign_id}&utm_content={ad_id}.{source_type}.{position_type}.{position}&utm_term={keyword}
```

Директ автоматически добавляет `yclid`. Сайт принимает URL с этими параметрами, сохраняет источник при переходах между страницами и прикладывает его к отправленной заявке.

## 5. Структура запуска

- География: Омск и Омская область.
- Разделить группы объявлений по товарным направлениям и вести каждую на соответствующую страницу.
- Не смешивать Поиск и РСЯ в одном первом тесте, чтобы отдельно видеть качество трафика.
- В объявлениях использовать реальные цены и характеристики с посадочной страницы.
- Подключить организацию из Яндекс Бизнеса, если карточка компании заполнена и контакты совпадают с сайтом.
- Включить мониторинг доступности сайта.
- Проверить быстрые ссылки: каталог направления, доставка, калькулятор, контакты.

## 6. Проверка перед оплатой

1. Открыть каждую рекламную ссылку с UTM-параметрами.
2. Отправить тестовую форму и тестовую корзину.
3. Проверить получение двух писем и рекламного блока в них.
4. Проверить появление целей в Метрике.
5. Только после этого запускать показы и оплачивать трафик.
